eBook-Export-Plugin-for-WordPress by chrisclarke

WordPress plugin to allow easy generation of ebooks in a variety of formats from the contents of a category. Intended mainly for web fiction but also useful for non-fiction works as well.

WordPress plugin to allow easy generation of ebooks in a variety of formats from the contents of a category. Intended mainly for web fiction but also useful for non-fiction works as well. Current Supported Output Formats: - EPUB - Plain Text (experimental) Planned Supported Output Formats: - HTML - HTML (Zipped) - MobiPocket - DocBook

Dependencies

WordPress >2.9 PHP 5.2 with Zip Support PHP exec and KindleGen (for MobiPocket support)

Install

Upload eBook-Export-Plugin-for-WordPress folder to your wp-content/plugins directory and activate in the WordPress admin panel.
